En el DOM existen dos niveles de exclusion:

-> Espacios y nuevas lineas antes de "head" son ignorados.

-> Todo elemento puesto despues de "/body" es automaticamente movido hacia dentro del cuerpo y puesto al final.

Autocorreccion
Si el navegador encuentra html malformados, automaticamente los corrige, aun cuando no exista la firma "html" explicitamente, siempre existira en el DOM y el navegador la creará, lo mismo sucede para "body".

Existen 12 tipos de nodos, en la prática usanlmente se usan 4:

1-> Documents: el punto de entrada de DOM

2-> Elemetos de nodos: Los hijos de Html (las tag)

3-> Comentarios: se puede poner informacion aqui, la cual no será mostrada, pero Js puede leerla del DOM

4->Nodos textos: contienen texto

Sumario

-> Las etiquetas se convierten en elementos de nodos y forman la estructura

-> Los textos se convierten en textos de nodos

-> Todo los elementos en Html tienen un lugar en el DOM
